
  ```
  这个算法使用了大量的额外空间，如何修改以降低空间复杂度？

  [在此粘贴代码]
  ```

## 系统性能优化

### 缓存机制

**Prompt 示例：**

- **实现缓存**

  ```
  如何在应用程序中添加缓存机制，以减少对数据库的频繁访问？

  [描述应用场景]
  ```

- **缓存失效策略**

  ```
  请解释常见的缓存失效策略，如 LRU、LFU，并建议在什么情况下使用哪种策略。
  ```


- **并行处理**

  ```
  如何利用多核 CPU，实现程序的并行处理，以提升性能？

  ```

  ```
  [在此粘贴 SQL 语句]

  ```
