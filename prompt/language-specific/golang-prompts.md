## 代码生成

### 函数与方法

**Prompt 示例：**

- **编写函数**

  ```
  请帮我编写一个函数，计算斐波那契数列的第 N 个数，并使用递归和迭代两种方式实现。
  ```

- **方法与接收者**

  ```
  请示范如何为自定义类型定义方法，包括值接收者和指针接收者的区别。
  ```

### 结构体与接口

**Prompt 示例：**

- **定义结构体**

  ```
  请创建一个名为 `Person` 的结构体，包含姓名和年龄字段，并实现一个方法用于输出自我介绍。
  ```

- **使用接口**

  ```
  请解释 Go 中的接口概念，并提供示例，展示如何实现多态。
  ```
