## 项目创建与配置

### 创建项目

**Prompt 示例：**

- **创建新项目**

  ```
  请告诉我如何使用 Django 创建一个新项目和应用程序，并简要解释每个命令的作用。
  ```

- **虚拟环境**

  ```
  在开始 Django 项目之前，如何创建和激活虚拟环境？请提供命令示例。
  ```

  ```

  ```
  我想将数据库从默认的 SQLite 更改为 PostgreSQL，应该如何修改 `settings.py` 文件？
